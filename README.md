# e-zpassin

## Description
e-zpassin is a lightweight and efficient expense tracking application designed for users who want a simple and organized way to record their daily expenses. The system focuses on usability, clarity, and practical functionality while offering a structured approach to managing financial entries. Users can add expenses, view a complete list, and generate receipt previews for recordkeeping. With its component-based structure, e-zpassin is easy to maintain, extend, and customize. The interface is clean and responsive, providing a smooth user experience on both desktop and mobile devices. Developers can easily contribute to the project thanks to its clear file organization, readable code, and detailed documentation. Whether for personal budgeting or small business use, e-zpassin offers an accessible solution for tracking spending consistently and accurately.

---

## Features
- Add expenses with name and amount
- Display a dynamic list of expenses
- Receipt preview section
- Clean and responsive user interface
- Easy-to-understand project structure
- Beginner-friendly and open for contributions
